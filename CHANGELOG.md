# Changelog

## February 20, 2026

### New Features
- **Metal Frame Sign post option** - New budget-friendly post option: angle iron frame, powder coated finish, fits 18"h x 24"w signs ($40)
- **Order editing** - Customers can now edit non-completed orders to add/remove riders, lockboxes, brochure boxes, and sign options without being double-charged for fuel surcharge
- **Order history redesign** - Order history now shows orders grouped with status badges (Pending, Confirmed, Scheduled, In Progress, Completed, Cancelled) instead of a flat item list
- **$40 no-post surcharge** - Orders without a post now include a $40 service trip fee, clearly shown in the order summary
- **Promo code fuel surcharge waiver** - Admins can now create promo codes that waive the $2.47 fuel surcharge (useful for fully-free promos)
- **Sign inventory always visible** - "Sign in inventory" option now always shows during ordering; displays a helpful error if no signs are in storage

### Improvements
- **Admin order confirmation emails** - Now include full installation details: property type, post type, installation location, gated community info, gate code, marker placement, sign orientation, special requests, and complete pricing breakdown
- **Step navigation in order wizard** - Visiting a step now keeps it active (pink) when navigating back; mobile step dots added for quick navigation between visited steps
- **"No Post Needed" repositioned** - Moved below post options with a divider for clearer visual hierarchy
- **Homepage CTA updated** - Subheader now reads "Next-day sign installation so you can focus on closing deals"

### Bug Fixes
- **Mobile checkout modal** - Card info modal no longer exceeds the viewport on small screens; content is now scrollable
- **Schedule a Trip menu** - "Other addresses" dropdown no longer overflows the viewport
- **Active installations action menu** - 3-dots menu replaced with a centered popup overlay that doesn't get cut off by table overflow
- **Reinstallation fee removed** - Reinstallation pricing has been fully removed from the system
